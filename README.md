# CBUILD

A quality of life tool to compile your c files with gcc by listing them in a txt file

## Usage

````
cbuild --out (prg_name)
````

*The --out opt is optional*

you will need to make a cbuild.txt file, that list all c files.

````
# And you can put comments !
main.c

# and empty lines !
foo.c
bar.c
````
